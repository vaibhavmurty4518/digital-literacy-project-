# Task 1: Digital Literacy Infographic

## Objective
To create an infographic explaining digital literacy for beginners.

## Description
This infographic highlights:
- What is digital literacy
- Useful digital tools (GitHub, LinkedIn, Google Docs, Gmail)
- Online safety tips (OTP safety, passwords, avoiding scams)

## Tools Used
- Canva / AI Generated Design

## Outcome
The infographic helps freshers understand how to use digital tools safely and effectively.
